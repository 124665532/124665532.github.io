String类

发表于 2016-05-07   |  

#include<iostream>
#include<stdlib.h>
#include<string>//是string而不是string.h
using namespace std;
int main() { string s1 = "123";//s1本身其实是一个对象 string s2("123"); cout<<s1<<s2<<endl; system("pause"); return 0; }


一、初始化string对象的方式
string s;//空串
string s(“ABC”);//与string s = “ABC”;效果相同
string s3(s2);//将s2复制给s3
string s(n,’c’);//s的内容是n个c，n是一个整数

二、常用操作 (由此可以看出来string类型是一个对象)
s.empty() 空串返回TRUE，否则返回FALSE
s.size() 返回长度
s[n] 像数组一样可以操作里面的字符
s1+s2 2个字符串连接
s1 = s2 把s2赋值给s1
v1 == v2 可以进行等号判断，想等返回True，否则返回False
v1 != v2 用法类比上

三、一些注意事项
string z = “1” + s;//True
string z = v + s;//True
string z = “1” + “2” //像这样是不可以的

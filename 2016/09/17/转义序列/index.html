#include<iostream>
using namespace std;
int main() { cout<<"\aOperation \"HyperHype\" is now activated!\n"; cout<<"Enter your agent code:______\b\b\b\b\b\b"; long code; cin>>code; cout<<"\aYou entered "<<code<<"...\n"; cout<<"\aCode verified!Proceed with Plan Z3!\n"; system("pause"); return 0; }

\b不是正真的退格，它只是将光标向前移动，如果之后有输入就会覆盖，如果没有就会保留原来的。
比如输出”asdf\b\b”那么你看到的任然是asdf但是光标在倒数第二个字符上.
但是如果你继续输出字符接下来的字符就会从倒数第二个字符开始覆盖.
“asdf\b\bc”输出的是ascf因为倒数第二个字符被覆盖但是倒数第一个未被覆盖
\a会响一下，蛮有意思的，下次做的时候可以用的上。

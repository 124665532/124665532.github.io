搭建静态博客

2016-10-04 • 标签: Git, Hexo

一、 前期准备

1. Github账号注册

2. Git for Windows下载

3. Node.js下载

二、 搭建静态博客

一、Git for windows 部署

1.设置用户名和邮箱

在git-bash中分别输入以下命令来设置：
git config –global user.name “yourname”
git config –global user.name “youremail”
这里的用户名以及邮箱和注册的时候保持一致。
设置完之后可以再分别输入以下代码来检验是否设置成功：
git config –global user.name
git config –global user.name

2.设置SSH KEY

在git-bash中输入：
ssh-keygen -t rsa -C yourmail@example.com
cd ~/.ssh
输入ls（显示当前路径下文件）后在输入：
vim id_rsa.pub
接着把里面的公钥（这个字读yue）复制到github的Settings->SSH and GPG keys中。

3.关于仓库

新建：在github上新建一个repository，命名为username.github.io。
克隆：在本地初始化仓库后（git init）输入：
Git add “html.zip”
Git commit –m “add a zip file”

二、Hexo部署

1.使用cmd输入：

npm install hexo -g

2.再选择合适路径新建目录并进入输入：

mkdir blog
cd blog

3.初始化hexo输入：

hexo init

4.启动hexo输入：

hexo server

此时进入localhost:4000应该可以打开默认主题的博客

5.修改站点配置文件：

type: git
repository: your SSH Github link generated in Github
branch: master

6.打开cmd安装hexo-deployer-git

npm install hexo-deployer-git -S

7.生成静态网页：

hexo g

8.部署到github上

hexo d

三、遇到的问题及其解决方法

1.hexo d步骤出现问题

在使用markdown语言的时候注意冒号后面要加上空格，不然不会识别。
而且type后的名字后面要用git而不是github
或者将node.js重新安装

2.关于markdown语言的编译器

http://www.cr173.com/soft/22141.html#address

3.关于公益404页面无法显示问题

1.由于github给的是二级域名无法使用，申请一个顶级域名就可以（土豪可以考虑）。
2.在知乎上面看到有大牛自己做了一个公益404页面，可以在二级域名使用，以下是链接：http://www.zhihu.com/question/21650209

4.关于博客中网易云音乐插入不显示

之前试了很多方法也没有成功，后来看了一个同学的博文说是js代码中不识别http而只识别https，所以只要在网易云音乐提供的js代码中将音乐网址的http后面加一个s就可以解决。

5.关于hexo中博文占主页过长问题

因为不进行手动处理默认是所有博文全部显示，后来查阅资料得知在markdown语言中只要插入

可以手动进行缩略。避免主页过长。

三、小结

这次跟着大神学长学习了搭建静态博客，总的来说还是比较顺利，也没有碰到什么特别卡手的问题。在博客的主题配置以及新功能的添加花了不少时间，希望能把这个曾经遗忘多年的BLOG重新拾起。这次最大的感受就是和以往不同的是，前端做起来非常有成就感，能做出实在的东西，这个能激发我不断学习的激情。希望在接下来几节课可以学习到更多的知识！

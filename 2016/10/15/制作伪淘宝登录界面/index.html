制作伪淘宝登录界面

发表于 2016-10-15   |  

一、前期准备

1.WebStorm下载
2.bootstrap-3.3.5-dist库的下载
3.HTML CSS JS相关知识的学习

二、学习制作伪淘宝登陆页面

一、创建项目

1.创建一个project命名为“Taobao”，并且将相关的素材以及bootstrap库放入文件夹下。

2.在项目文件夹下新建文件“index.html”为主页面。以及“style.css”为css样式的文件。（理论上CSS文件可以不用，为了代码的简洁性，将样式的设置单独放在一个文件内）并使用：

<link href="bootstrap-3.3.5-dist/css/bootstrap.min.css" rel="stylesheet">
<link href="styles.css" rel="stylesheet">

将相关库与文件链接。
注：在这里如果将2句话对调后面会出现样式错误，CSS样式有先后的，后来的会覆盖之前的。

二、开始写HTML、CSS

1.使用div标签将网页分为“top”，“mid”，“bottom”，使用类名class。（注：class与id的区别。Class可以被多个标签使用，而id每个标签只有唯一一个。）

2.在top标签中插入a标签超链接并且用img将文件夹中的logo图片引用。同时在css文件中设置格式。

3.Css设置格式时候需要运用盒子模型，并通过其将logo放入合适位置。

4.在此使用div标签在mid类中插入登陆类：login并对其CSS进行设置。并使用bootstrap库中提供的登录框样式在login类中插入和设置input标签。

<input type="text" id="u" class="form-control" value="手机号/会员名/邮箱"     onfocus="if(this.value==defaultValue) {this.value='';this.type='text'}"     onblur="if(!value) {value=defaultValue; this.type='text';}" style="color:#CCC;">

再使用以上代码将用户名框后插入初始值。

5.使用button标签做出登陆按钮。其样式参考淘宝网。
并且在css文件中使用.类名:hover来设置鼠标移动到按钮上之后的按钮颜色变化。
（注： 是占位符，相当于一个空格，因为在js中多个空格只有一个空格效果）

6.再次使用a标签插入忘记密码和注册账号。

<a href="#" id="forget">忘记密码</a>
<a href="#" id="reg">免费注册</a>

注：关于超链接下划线的问题，可以在css中使用

text-decoration: none;

来消除掉。

7.同理在bottom中使用各种a标签插入超链接，再用css设置样式，比如：

<a href="http://www.alibabagroup.com/cn/global/home" style="font-size:12px">阿里巴巴集团</a> |
<a href="http://www.alibaba.com/?spm=a2107.1.1997523009.2.OQvvDS" style="font-size:12px">阿里巴巴国际站</a> |
<a href="http://www.1688.com/?spm=a2107.1.1997523009.3.OQvvDS" style="font-size:12px">阿里巴巴中国站</a> |

三、JS的插入

1.JS可以插入在html的各个位置，只要使用script标签即可。我觉得将他放在html 的文件底部。

2.使用相关代码获取id信息

document.getElementById(id)

为了方便起见写了如下代码：

var $ = function(id){
    return document.getElementById(id);
}

这样可以用$来代替一长串的字母。

3.对登陆框进行简单的逻辑判断，在这里我和任务要求所做的有所差距，考虑到和淘宝 原网页的相似性，没有使用正则表达式等11位手机号的判断，而是直接使用：

if(u.value ===""||u.value ==="手机号/会员名/邮箱"||p.value===""){
    $("mmdl").style.display="none"
    $("ban1").style.display="block"
    setTimeout("back()", 3000)
}

其中back函数是使样式恢复。
在这里我使用了标签的隐藏于显示来保证与官网的一致性。即：

style="display: none"

（注：display隐藏和visibility隐藏的区别。如果使用display隐藏，不会占据空 间，而visibility隐藏仍然会占据一定的空间）

三、遇到的问题及其解决方法

1.登陆框样式蜜汁错误

<link href="bootstrap-3.3.5-dist/css/bootstrap.min.css" rel="stylesheet">
<link href="styles.css" rel="stylesheet">

以上代码不能调动位置即可，CSS样式有先后的，后来的会覆盖之前的。

2.Js代码在登录框中无法正常使用页面总刷新

<button class="btn" style="color:white;font-weight:bold;" value="submit" id="submit" type="button">登 &nbsp;录</button>

其中的style改成 button而不是submit，否则网页会刷新，之后的js代码将无法使用。

3.标签隐藏仍然占空间
之前在百度上寻找的visibility设置来隐藏发现其虽然隐藏了标签，但是仍然会占用空间，因此采用display来隐藏标签，这样就不会占用一点的空间啦~

4.关于JS代码中仍然没有解决的问题
由于时间有限，水平有限。setTimeout函数研究的并不是很清楚，因此登陆后的0.2秒延时没有做出来，只做出了3秒后标签恢复原状。
还有JS语言不熟，对于含参的函数无法熟练使用，导致了最后的恢复标签样式的函数不能带有参数，只能写多个同样的函数。（在这里被自己蠢哭了/(ㄒoㄒ)/~~）

var back = function(){
    $("mmdl").style.display="block"
    $("ban1").style.display="none"
}
var back2 = function(){
    $("mmdl").style.display="block"
    $("ban2").style.display="none"
}
var back3 = function(){
    $("mmdl").style.display="block"
    $("ban3").style.display="none"
}
